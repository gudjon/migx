## One-line

NextGen module order is **filesystem/library → load → deck A → waveform → deck B → mixer/crossfader**, then prep chips / co-pilot — matching the irreducible DJ loop and what Serato/rekordbox/djay train users to expect.

## Needs (N0–N6 = “can mix”)

Owned files → find → load → two decks → EQ/cue/crossfader → master out. AI only after that loop is green.

## Leader UI loves to steal (not clone)

- Dual-deck + center mixer literacy (everyone)
- Clean Serato-like performance chrome; progressive depth
- Waveforms with cues + colour; **vocal markers** (rekordbox love)
- Crates + search that actually work (Serato 4.0 catch-up story)
- Stems as expected (quality bar later)
- Innovation reference: djay + VirtualDJ; club base: rekordbox + Serato
